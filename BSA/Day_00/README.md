# Day_00 — Базовые артефакты анализа: источники, глоссарий, декомпозиция

В этом дне создаются первые фундаментальные артефакты, на которых дальше строится весь проектный анализ. Мы начинаем работу над двумя кейсами:

- **Barbershop Appointment Project (BAP)** — система онлайн-записи в барбершоп;  
- **Delivery Order System (DOS)** — система курьерской доставки заказов.

**Фокус дня** — заложить основу для аналитической части проекта: сформировать рабочее пространство, собрать информацию, выровнять терминологию и научиться применять правила декомпозиции на простых примерах.
