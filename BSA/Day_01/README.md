# Day_01 — Заинтересованные стороны и их потребности

В этом дне продолжаем работу с двумя кейсами:

- Barbershop Appointment Project (BAP),
- Delivery Order System (DOS).

**Фокус дня** — научиться:

- фиксировать стейкхолдеров в виде каталога;
- раскладывать их по слоям «луковичной диаграммы»;
- описывать интересы, потребности и проблемы разных ролей;
- обновлять глоссарий с учётом найденных ролей и терминов.

---

## Упражнение 00 — Каталог заинтересованных сторон (DOS)

**Задача:**  
Собрать реестр заинтересованных сторон **только для кейса DOS**.  
Нужно перечислить ключевые группы и отдельных участников (курьеры, операторы, диспетчеры, администраторы, бухгалтерия, заказчики и т.д.), задать для них основные характеристики: роль, тип участника, уровень взаимодействия с системой, влияние на проект.

**Результат:**

- `ex00_DOS_stakeholders.xlsx`

## Упражнение 01 — Луковичная диаграмма стейкхолдеров (DOS)

**Задача:**  
Построить луковичную диаграмму **только для кейса DOS** на основе реестра из упражнения 00:

- выделить слои в зависимости от близости к системе (ядро — система, далее пользователи, внешние участники, контур регуляторов и т.д.);
- разнести по слоям роли и/или группы стейкхолдеров;
- при необходимости добавить недостающие роли и вернуть их в реестр стейкхолдеров.

## Упражнение 02 — Интересы, потребности и проблемы стейкхолдеров

**Задача:**  
Для **каждого кейса (BAP и DOS)** оформить таблицу с интересами, потребностями и проблемами ключевых стейкхолдеров, включая внешних участников:

- зафиксировать, что важно для каждой роли;
- какие у неё ожидания к системе;
- какие текущие «боли» система должна закрыть.

Формат — табличный, пригодный для дальнейшей работы с требованиями.

**Результат:**

- `ex02_DOS_needs.xlsx`

## Упражнение 03 — Актуализация глоссария

**Задача:**  
Дополнить глоссарий по обоим кейсам новыми терминами, которые появились при работе со стейкхолдерами:

- новые роли (например, диспетчер, оператор, владелец сервиса);
- типы взаимодействий (доставка, переназначение заказа, расчёт вознаграждения и т.п.);
- объекты предметной области, связанные с доставкой и записью в барбершоп.

Для каждого термина нужно зафиксировать определение и источник (если есть).
